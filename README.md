# sorting_visualizer
a simple sorting visualizer using c++\sfml library

## it contains two types of sorting algorithms:
1- bubble sort<br/>
2- selection sort<br/>

## another sorting algorithms will be added in the future:
1- merge sort<br/>
2- quick sort<br/>
3- insertion sort<br/>
